📊 E-Commerce Sales and Customer Insights Dashboard

📖 Introduction
This project presents an interactive **E-Commerce Sales Dashboard** built using Power BI.  
It analyzes sales data to provide insights into business performance, customer behavior, and product trends.

---

🎯 Objectives
- Analyze overall sales performance  
- Understand customer purchasing behavior  
- Identify top-performing products and cities  
- Track sales trends over time  

---

📊 Key KPIs
- 💰 Total Sales: **2M**  
- 📦 Total Orders: **80**  
- 🛒 Total Quantity: **252**  
- 👥 Unique Customers: **10**  

---

📈 Visualizations
- 📉 Line Chart → Sales Trend Over Time  
- 🍩 Donut Chart → Sales by Category  
- 📊 Bar Chart → Sales by Product  
- 🌍 Bar Chart → Sales by City  
- 📋 Table → Customer-wise Sales  

---

🎛 Features
  - Interactive filters:
  - Category  
  - City  
  - Year  
  - Month  
- Clean and user-friendly dashboard design  
- Dynamic data exploration  

---

🛠 Tools Used
- Power BI  
- Excel Dataset  

---

🧩 Dataset Fields
- Order ID  
- Order Date  
- Customer Name  
- Product  
- Category  
- City  
- Quantity  
- Total Sales  

---

🔍 Key Insights
- Sales show fluctuations with peak periods  
- A few products generate the majority of revenue  
- Cities like Bangalore contribute higher sales  
- Customer base is small but valuable  

---

🚀 Future Scope

## ✅ Conclusion
The dashboard provides a clear and interactive view of sales performance and customer insights.  
It helps businesses make better decisions and improve overall growth.
