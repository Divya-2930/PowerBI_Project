📊 E-Commerce Sales Dashboard (Power BI)
📖 Introduction
This project showcases an interactive E-Commerce Sales Dashboard built using Power BI. It provides insights into sales performance, customer behavior, and product trends through     dynamic and visually appealing reports.

🎯 Objectives

Analyze sales performance

Understand customer behavior

Identify top products and cities

Track trends over time

📊 Key KPIs

💰 Total Sales

📦 Total Orders

🛒 Total Quantity

👥 Unique Customers

📈 Visualizations

Line Chart → Sales Trend

Donut Chart → Sales by Category

Bar Chart → Sales by Product

Bar/Map → City-wise Sales

🎛 Features

Interactive filters (Category, City, Year, Month)

User-friendly dashboard layout

Real-time data exploration

🛠 Tools Used

Power BI

Excel / CSV

🔍 Key Insights

Sales vary across time and regions

Few products drive most revenue

High-performing cities contribute significantly

Customer insights help improve strategy

🚀 Future Enhancements

Real-time data integration

Predictive analytics

Customer segmentation

Advanced KPI tracking

✅ Conclusion

This dashboard helps businesses make informed decisions by providing clear insights into sales and customer data. It is a powerful tool for improving performance and strategy.
